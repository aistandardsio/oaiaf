## [v0.1.0] - 2026-06-29 ([`515bf7e`](https://github.com/aistandardsio/oaiaf/commit/515bf7e))

### Highlights

- Initial release of the Open Agentic Identity and Authorization Framework (OAIAF)
- Five-layer identity stack: Lifecycle, Workload, Agent Auth, Human Delegation, Authorization
- Go SDK with providers for ID-JAG, AAuth, and AIMS (SPIFFE-based agent identity)
